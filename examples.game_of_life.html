class examples.game_of_life.simulation.GosperGliderGun¶

Bases: adjsim.core.VisualSimulation

Implementation of a gosper glider gun pattern starting condition.

This was the first discovered pattern to exhibit infinite growth. It was a submission by Bill Gosper in response to a $50 contest to prove/disprove Conway’s conjecture that there exist patterns that grow infinitely. Description can be found at http://www.conwaylife.com/w/index.php?title=Gosper_glider_gun.
